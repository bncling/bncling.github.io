Affine varieties

Definition Let \(k\) be a field and let \(f_1, \dots, f_s\) be polynomials in the ring \(\pring{k}{x_1, \dots, x_n}\). Then the affine variety defined by \(f_1, \dots, f_s\), denoted \(\V(f_1, \dots, f_s\)) is the set \[\V(f_1, \dots, f_s) = \{(a_1, \dots, a_n) \in k^n \mid f_i(a_1, \dots, a_n) = 0, i = 1, \dots, s\},\] that is, the set of points in the affine space over \(k\) at which all of the polynomials vanish simultaneously....

Ideals of a ring
