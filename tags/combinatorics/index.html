Bell numbers

Definition The \(n\)th Bell number, denoted \(B_n\) is the total number of ways of partitioning the set \([n]\). Properties The Bell numbers are given by the formula \[B_n = \sum_{k=0}^n \stirlingII{n}{k}\] where \(\stirlingII{n}{k}\) is the Stirling number of the second kind which counts the number of ways to partition \([n]\) into \(k\) nonempty parts. Permutations
