Affine subsets

Definition An affine subset of a vector space \(V\) is a subset of the form \(v + U\) where \(v \in V\), \(U\) is a subspace of \(V\), and \[v + U = \{v + u \mid u \in U\}\] is the sum of the vector \(v\) and the space \(U\). Linear maps

Also called linear transformations. Definition A linear map from a vector space \(V\) to a vector space \(W\) is a function \(T: V \to W\) with the following properties: additivity: \(T(u + v) = T(u) + T(v)\) for all \(u,v \in V\); homogeneity: \(T(av) = aT(v)\) for all \(v \in V\) and \(a \in \F\). Vector spaces
