Completely normal spaces

Definitions Two sets \(A\) and \(B\) in a topological space are separated if and only if \(\overline{A} \cap B = \emptyset\) and \(A \cap \overline{B} = \emptyset\). A topological space \((X, \T)\) is completely normal if and only if for any two separated sets \(A\) and \(B\), there exist open sets \(U\) and \(V\) such that \(A \subset U\) and \(B \subset A\).
