Definition Let \(X\) be a set totally ordered by a relation \(\leq\). Closed sets

Definition In a topological space \((X, \mathcal{T})\), a closed set \(A\) is a subset of \(X\) that is equal to its closure (\(\overline{A} = A\)). That is, \(A\) contains all its limit points.
