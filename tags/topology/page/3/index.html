Closed sets

Definition In a topological space \((X, \mathcal{T})\), a closed set \(A\) is a subset of \(X\) that is equal to its closure (\(\overline{A} = A\)). That is, \(A\) contains all its limit points. Properties of closed sets Theorem 2.14: A set \(A\) is closed if and only if its complement \(X - A\) is open.
