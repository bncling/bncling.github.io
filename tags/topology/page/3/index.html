Finite complement topology

Basically what the name makes it sound like. Similar in definition to the countable complement topology. Definition Let \(X\) be a set. A set \(U\) is in the finite complement topology if and only if \(X - U\) is finite or if \(U = \emptyset\). This topology is also called the cofinite topology. Indiscrete topology

The coarsest possible topology on a set \(X\). Definition Let \(X\) be a set. The indiscrete topology is \(\mathcal{T} = \{\emptyset, X\}\). The topological space \((X, \{\emptyset, X\})\) is called an indiscrete space. Properties By definition, the only sets in the indiscrete topology that are open are \(X\) and \(\emptyset\), and these are also the only closed sets. K-topology

Denoted \(\Rh\) in the textbook I’ve been using, the K-topology is a topology on \(\mathbb{R}\). Definition Let \(K = \{1/n\}_{n \in \mathbb{N}}\) be the harmonic sequence. The K-topology on \(\mathbb{R}\) is generated by the basis containing sets of the form \((a,b)\) and sets of the form \((a,b) - K\) where \(a, b \in \mathbb{R}\) with \(a < b\). Properties \(\Rh\) is finer than the standard topology. The discrete topology

The discrete topology is the finest topology on a given set. A set together with the discrete topology is called a discrete space. Definition Let \(X\) be a set. Then the discrete topology on \(X\) contains all subsets of \(X\), that is, the topology is \(\mathcal{T} = 2^X\), the power set of \(X\).
