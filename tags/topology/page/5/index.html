Closed sets

Definition In a topological space \((X, \mathcal{T})\), a closed set \(A\) is a subset of \(X\) that is equal to its closure (\(\overline{A} = A\)). That is, \(A\) contains all its limit points. Isolated points

Properties If \(p\) is an isolated point of a set \(A\) in a topological space \((X, \mathcal{T})\), then there exists an open set \(U\) such that \(U \cap A = \{p\}\).
