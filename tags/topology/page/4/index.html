Indiscrete topology

The coarsest possible topology on a set \(X\). Definition Let \(X\) be a set. The indiscrete topology is \(\mathcal{T} = \{\emptyset, X\}\). The topological space \((X, \{\emptyset, X\})\) is called an indiscrete space. Properties By definition, the only sets in the indiscrete topology that are open are \(X\) and \(\emptyset\), and these are also the only closed sets. Normal spaces

Definition Let \((X, \T)\) be a topological space. Regular spaces

Definition Let \((X, \T)\) be a topological space. It is regular if and only if for every point \(x \in X\) and closed set \(A \subset X\) with \(x \notin A\), there exist disjoint open sets \(U\) and \(V\) such that \(x \in U\) and \(A \subset V\). The discrete topology

The discrete topology is the finest topology on a given set. A set together with the discrete topology is called a discrete space. Definition Let \(X\) be a set. Then the discrete topology on \(X\) contains all subsets of \(X\), that is, the topology is \(\mathcal{T} = 2^X\), the power set of \(X\). The box topology

Another, weirder topology on an infinite product. Definition Let \(\{(X_\alpha, \T_\alpha)\}_{\alpha \in \lambda}\) be a (possibly infinite) collection of topological spaces. Closed sets

Definition In a topological space \((X, \mathcal{T})\), a closed set \(A\) is a subset of \(X\) that is equal to its closure (\(\overline{A} = A\)). That is, \(A\) contains all its limit points. Properties of closed sets Theorem 2.14: A set \(A\) is closed if and only if its complement \(X - A\) is open. Isolated points
