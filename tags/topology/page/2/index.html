Path connectedness

Definitions Let \((X,\T)\) be a topological space. A path from \(x\) to \(y\) for points \(x,y \in X\) is a continuous function \(f: [0,1] \to X\) such that \(f(0) = x\) and \(f(1) = y\). A space \(X\) is path connected if and only if for all \(p,q \in X\) there exists a path from \(p\) to \(q\). Properties Theorem 8.35: All path connected spaces are connected.
