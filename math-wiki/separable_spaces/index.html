Separable spaces

December 22, 2021

Defintion

A topological space \((X, \T)\) is separable if and only if \(X\) has a countable dense subset.

Properties

Exercise 5.4: A space \((X, \T)\) being separable does not imply that all of its subspaces are separable.

Theorem 5.5: If two spaces \(X\) and \(Y\) are separable, then \(X \times Y\) with the product topology is separable.

Examples

The space \(\Rs\) is separable, but \(\R\) with the discrete topology is not separable because the reals are uncountable and so for any countable subset, there is point not in the subset, and the set containing this point is open in the discrete topology, meaning not all open sets contain a point in the countable subset and so no countable subset can be dense.
