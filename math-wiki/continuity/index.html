Continuity

February 9, 2022

Continuous functions can be drawn without picking up your pencil…?

The \(\varepsilon\)-\(\delta\), limit related definition

A function \(f: \mathcal{D} \to \mathbb{R}\) is continuous at a point \(c \in \mathcal{D}\) if and only if the limit of \(f\) at \(c\) exists and \(\lim \limits_{x \to c} f(x) = f(c)\). That is, for every \(\varepsilon > 0\), there exists a \(\delta > 0\) such that \(\vert x - c \vert < \delta\) implies \(\vert f(x) - f(c) \vert < \varepsilon\). A function \(f: \mathcal{D} \to \mathbb{R}\) is continuous if and only if it is continuous at the point \(c\) for all \(c \in \mathcal{D}\).

The topological definition

A function \(f: X \to Y\) for topological spaces \(X\) and \(Y\) is continuous if and only if for every \(V\) open in \(Y\), the preimage \(f\inv(V)\) is open in \(X\). A function \(f: X \to Y\) is continuous at a point \(x \in X\) if and only if for every \(V\) open in \(Y\) and containing \(f(x)\), there exists a \(U\) open in \(X\) that contains \(x\) such that \(f(U) \subset V\).

Properties

Theorem 7.1: The following are all equivalent characterizations of continuity for maps \(f: X \to Y\) between topological spaces \(X\) and \(Y\):

  • For every closed set \(K\) in \(Y\), the inverse image \(f\inv(K)\) is closed in \(X\).
  • For every limit point \(p\) of a set \(A\) in \(X\), the image \(f(p) \in \overline{f(A)}\).
  • For every \(x \in X\) and open set \(V\) with \(f(x) \in V\), there exists an open set \(U\) containing \(x\) such that \(f(U) \subset V\).

Theorem 7.5: The \(\varepsilon\)-\(\delta\) definition given above is equivalent to the topological definition for maps \(f: \Rs \to \Rs\) where \(\Rs\) is the set of real numbers with the standard topology.

Theorem 7.6: If \(X\) is 1st countable, then a map \(f: X \to Y\) for some topological space \(Y\) is continuous if and only if for every convergent sequence \(x_n \to x\) in \(X\), the sequence \((f(x_n))_{n \in \N}\) converges to \(f(x)\) in \(Y\). Note that becuase \(\Rs\) is 1st countable, this gives another equivalent characterization of continuity for functions \(f: \Rs \to \Rs\).

Theorem 7.7: If \(f: X \to Y\) is continuous and \(D\) is dense in \(X\), then \(f\) is determined by the values it takes on \(X\).

Theorem 7.9: The composition of continuous functions is continuous.

The pasting lemma: If \(f:A \to Y\) and \(g:B \to Y\) are continuous and \(A\) and \(B\) are either both open or both closed in the space \(X = A \cup B\), then \(h: X \to Y\) is continuous where \(h = f\) on \(A\) and \(h = g\) on \(B\).

Theorems 7.13 and 7.14: To show \(f: X \to Y\) is continuous, it suffices to check that all preimages of basic or subbasic open sets in \(Y\) are open in \(X\).

Theorem 7.24: If \(f: X \to Y\) is continuous with domain \(X\) normal and codomain \(Y\) Hausdorff, then \(f\) is a closed map.

Theorem 7.40: Let \(Z\) be a topological space and let \(\prod_{\alpha \in \lambda} X_\alpha\) be the product of spaces \(X_\alpha\) equipped with the product topology. Then any function \(g: Z \to \prod_{\alpha \in \lambda} X_\alpha\) is continuous if and only if \(\pi_\beta \circ g\) is continuous for all projection functions \(\pi_\beta\), \(\beta \in \lambda\).

Properties preserved by continuous functions

Let \(X\) and \(Y\) be topological spaces and let \(f: X \to Y\) be continuous and surjective. Then by theorems 7.15 through 7.25:

  • \(X\) compact \(\implies\) \(Y\) compact;
  • \(X\) Lindelöf \(\implies\) \(Y\) Lindelöf;
  • \(X\) countably compact \(\implies\)\(Y\) countably compact;
  • \(D\) dense in \(X\) \(\implies\) \(f(D)\) dense in \(Y\);
  • \(X\) separable \(\implies\) \(Y\) separable;
  • \(X\) normal and \(f\) closed \(\implies\) \(Y\) normal;
  • \(X\) has basis \(\{B_\alpha\}_{\alpha \in \lambda}\) and \(f\) open \(\implies\) \(\{f(B_\alpha)\}_{\alpha \in \lambda}\) a basis for \(Y\);
  • \(X\) 2nd countable and \(f\) open \(\implies\) \(Y\) 2nd countable;
  • \(X\) 2nd countable and compact, \(Y\) Hausdorff \(\implies\) \(Y\) 2nd countable;

Examples

Every rational function is continuous at every point in its domain.

Thomae’s function is continuous at every irrational point and discontinuous at every rational point.

Some maps between topological spaces are always continuous. For example, the constant map \(f: X \to Y\) defined by \(f(x) = y_0\) for some \(y_0 \in Y\) is continuous; if \(A \subset X\) and \(f\) is continuous, then the restriction map \(f\vert_A: A \to Y\) defined by \(f\vert_A(a) = f(a)\) is also continuous; and if we assume \(X \subset Y\), then the inclusion map \(i: X \to Y\) defined by \(i(x) = x\) is continuous.
