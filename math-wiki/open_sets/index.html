Open sets

July 12, 2021

Definition

In a topological space \((X, \mathcal{T})\), an open set \(U\) is a set with \(U \in \mathcal{T}\).

Properties of open sets

  • Theorem 2.3: A set \(U\) is open if and only if for each point \(p \in U\), there exists an open set \(U_p\) such that \(p \in U_p \subset U\).
  • If \(\{U_\alpha\}_{\alpha \in \lambda}\) is a collection of open sets, then \(\bigcup_{\alpha \in \lambda} A_\alpha\) is open, but \(\bigcap_{\alpha \in \lambda} A_\alpha\) need not be open. This is part of the definition of a topology.
  • If \(U\) and \(V\) are open, then \(U \cap V\) is open.
  • Theorem 2.14: The complement \(X - U\) of an open set is closed.
