Normal spaces

July 9, 2021

Definition

Let \((X, \T)\) be a topological space. It is normal if and only if for every pair of disjoint, closed sets \(A\) and \(B\), there are disjoint open sets \(U\) and \(V\) such that \(A \subset U\) and \(B \subset V\).

Properties

Theorem 4.9: A space \((X, \T)\) is normal if and only if for every closed set \(A \subset X\) and open set \(U\) containing \(A\), there exists an open set \(V\) such that \(A \subset V\) and \(\overline{V} \subset U\).

Theorem 4.10: A space \((X, \T)\) is normal if and only if for each pair of disjoint closed sets \(A\) and \(B\), there exist open sets \(U\) and \(V\) such that \(A \subset U\), \(B \subset V\), and \(\overline{U} \cap \overline{V} = \emptyset\). (That is, not only can disjoint open sets \(U\) and \(V\) be found separating \(A\) and \(B\), open sets with disjoint closures can be found.)

The Incredible Shrinking Theorem: A space \((X, \T)\) is normal if and only if for each pair of open sets \(U\) and \(V\) such that \(U \cup V = X\), there exist open sets \(U'\) and \(V'\) such that \(\overline{U'} \subset U\), \(\overline{V'} \subset V\), and \(U' \cup V' = X\). (That is, in a normal space, given two sets whose union is the whole space, two smaller sets can be found whose union is also the whole space.)

If a space \((X, \T)\) is \(T_4\), it is also normal by definition.

Being normal does not necessarily imply being \(T_3\), regular, Hausdorff, or \(T_1\).
