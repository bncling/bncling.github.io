Closed functions

February 1, 2022

Definition

A function \(f: X \to Y\) for topological spaces \(X\) and \(Y\) is called closed if and only for every closed set \(A \subset X\), the image \(f(A)\) is closed in \(Y\).

Properties

Theorem 7.21: If \(X\) is normal and \(f: X \to Y\) is continuous, surjective, and closed, then \(Y\) is normal.
