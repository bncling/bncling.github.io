July 5, 2021

Intuitively, “\(\lim \limits_{x \to a} f(x) = L\)” means that the outputs of a function \(f\) approaches the value \(L\) as the inputs approach the value \(a\).

Limits of sequences in topology

In a topological space \((X, \mathcal{T})\), we say that a sequence \((x_i)_{i \in \mathbb{N}}\) converges to \(p \in X\) or that \(p\) is the limit of the sequence if and only if for every open set \(U\) with \(p \in U\), there exists an \(N \in \mathbb{N}\) such that \(i > N\) implies that \(x_i \in U\).

Examples

  • If \(h(x) = \frac{x^2 - x - 2}{x - 2}\), then \(\lim \limits_{x \to 2} h(x) = 3\).
  • If \(f(x) = \frac{\sin x}{x}\), then \(\lim \limits_{x \to 0} f(x) = 1\).
