Power sets

December 21, 2021

Definition

Let \(A\) be a set. Its power set, denoted \(2^A\), is the set of all subsets of \(A\).

Properties

Theorem 1.18: If the set \(A\) is finite, then its power set has cardinality \(\vert 2^A \vert = 2^{\vert A \vert}\).

Theorem 1.19: For any set \(A\), there exists an injection from \(A\) into \(2^A\).

Theorem 1.20: The cardinality of \(2^A\) is the same as the cardinality of the set of all functions \(f: A \to \{0,1\}\).

Theorem 1.22: For all sets \(A\), there does not exist a surjection \(f: A \to 2^A\).
