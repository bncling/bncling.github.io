Intermediate Value Theorem (IVT)

July 5, 2021

Theorem statement

Let \(f: [a,b] \to \mathbb{R}\) be continuous and let \(L \in \mathbb{R}\) be a number strictly between \(f(a)\) and \(f(b)\). Then there exists a \(c \in (a,b)\) such that \(f(c) = L\).

Lemma

Proof

Without loss of generality, assume \(f(a) < f(b)\). Let \[A = \{d \in [a,b] \mid \text{for all } x \in [a,d], f(x) < L\}.\] Then \(a \in A\), so the set is nonempty, and \(b\) is an upper bound of \([a,b]\) so it is also an upper bound of \(A \subset [a,b]\). By the completeness axiom, \(A\) has a supremum, say \(c = \sup A\). By the lemma, there is a \(\delta_1 > 0\) such that \(f(x) < L\) for all \(x \in [a, a + \delta_1)\) and a \(\delta_2 > 0\) such that \(f(x) > L\) for all \(x \in (b - \delta_2,b]\). Then \(a < c < b\). Suppose for contradiction that \(f(c) < L\) so that by the lemma there exists a \(0 < \delta_3 < b-c\) such that \(f(x) < L\) whenever \() > L\). By the lemme there exists a \(0 < \delta_4 < c-a\) such that \(f(x) > L\) whenever \()\) so \(f(x) < L\), a contradiction. Therefore \(f(c) = L\).
