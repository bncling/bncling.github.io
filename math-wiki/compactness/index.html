Compactness

January 30, 2022

Definitions

Let \((X,\T)\) be a topological space and with a subset \(A \subset X\). A collection \(\Cov = \{C_\alpha\}_{\alpha \in \lambda}\) is called a cover of \(A\) if and only if \(A \subset \bigcup_{\alpha \in \lambda} C_\alpha\). The collection \(\Cov\) is called an open cover if each \(C_\alpha\) is an open set. A collection \(\Cov'\) is a subcover of \(\Cov\) if it is a subset of \(\Cov\) and is also a cover of \(A\).

A space \(X\) is called compact if and only if every open cover of \(X\) has a finite subcover.

A collection of sets has the finite intersection property if and only if all finite subcollections have nonempty intersections.

Properties

Theorem 6.1: All finite topological spaces are compact.

All compact spaces are Lindelöf and countably compact.

Theorem 6.3: If \(X\) is a compact space, then every infinite subset of \(X\) has a limit point.

Theorem 6.5: An equivalent characterization of a space \(X\) being compact is that all collections of closed sets with the finite intersection property have nonempty intersection.

Theorem 6.8: If \(X\) is compact, then closed subsets of \(X\) are compact.

Theorem 6.9: If \(X\) is Hausdorff, then compact subsets of \(X\) are closed.

Exercise 6.7: The union of finitely many compact sets is compact.

Exercise 6.11: If \(X\) is Hausdorff, then the intersection of an arbitrary number of compact sets is compact.

Theorem 6.12: If \(X\) is compact and Hausdorff, then \(X\) is also normal.

Theorem 6.13: Let \(\B\) be a basis for a space \(X\). Then \(X\) is compact if and only if every basic open cover has a finite subcover.

Alexander Subbasis Theorem: Let \(\Sub\) be a subbasis for a space \(X\). Then \(X\) is compact if and only if ever subbasic open cover has a finite subcover.

Tychonoff’s Theorem: If \(X_\alpha\) is compact for \(\alpha \in \lambda\), then the space \(\prod_{\alpha \in \lambda} X_\alpha\) with the product topology is compact.

Examples

In \(\Rs\), the Heine-Borel Theorem says that a subset \(A \subset \R\) is compact if and only if it is closed and bounded, and this extends to \(\Rs^n\) as well.
