Cyclic groups

Definitions A group \((G, \cdot)\) is a cyclic group if there exists an element \(a \in G\) such that every element of \(G\) can be written as \(a^k\) for some \(k \in \Z\). Dense subsets

Definition A subset \(A \subset X\) in a topological space \((X, \T)\) is dense in \(X\) if and only if \(\overline{A} = X\), where \(\overline{A}\) is the closure of \(A\) in \((X, \T)\). Properties Exercise 5.1 A subset \(A \subset X\) is dense in \(X\) if and only if every open set in \((X, \T)\) contains a point in \(A\). Fineness and coarseness

Given two topologies on a set \(X\), we can compare them with the notions of fineness and coarseness. You can think about the fineness of a topology determining the amount you can see (you “see” the open sets) when looking at the set \(X\). Group automorphisms

Definition An automorphism of a group \((G, \cdot)\) is a map \(f: G \to G\) that is an ismorphism. An inner automorphism of \(G\) is an automorphism \(i_g: G \to G\) given by conjugation: \(i_g (x) = gxg\inv\). Properties Because automorphisms are bijections from the set \(G\) to itself, the set of all automorphisms, denoted \(\Aut (G)\), is a subset of \(\Sym(G)\), the symmetric group on \(G\).
